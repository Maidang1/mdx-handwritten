# Keep AI generation explicit, reviewed, and upstream

Optional AI remains in an author-invoked tool upstream of the pure Scene Module: it may receive only disclosed content, produce an untrusted Scene proposal, and materialize a Reviewed plan artifact only after human approval and full `createScenePlan` validation. The tool stores that closed JSON artifact under an explicit opaque plan reference in source control, while private Review records own provider and disclosure evidence; ordinary builds only revalidate deterministic local inputs and never call a model, inspect a Review store, relocate stale ranges, or silently replace a missing, invalid, incompatible, or Stale Scene plan with newly inferred meaning. This trades automatic recovery for review integrity, reproducible offline builds, and a narrow privacy boundary.
